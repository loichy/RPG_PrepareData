# RPG_PrepareData

This repository contains R scripts to **prepare RPG data**. The main objectives are: 

  1. Download all RPG data files at the parcel level, for each year an each region
  2. Aggregate each region-year datafile at a commune-level or small region-level
  3. Assemble all region-year datafiles in a single panel dataset
  
To understand better the data processing steps open the RMarkdown file `RPG_PrepareData.Rmd`.
